MCDB API

静态 JSON 托管于本站；模糊搜索与分类浏览由 search.mcdb.astral.fan 提供。

汉化 I18n

  • GET /api/v1/manifest.json
  • GET /api/v1/i18n/{hex}.json（示例：Sodium → 4141）

分片规则：hex = utf8(id[0:2]).hex()，按需 GET 后取 [id]。

// Dart
final hex = utf8.encode(id.substring(0, 2)).map((b) => b.toRadixString(16).padLeft(2, '0')).join();
// GET https://mcdb.astral.fan/api/v1/i18n/$hex.json

标题索引

  • GET /api/v1/titles/manifest.json
  • GET /api/v1/titles/types.json — 分类目录与条数
  • GET /api/v1/titles/titles.pack.json — 全量并行数组（约 11MB，供 Worker 拉取）

模糊搜索

POST https://search.mcdb.astral.fan/v1/search
Content-Type: application/json

{"q":"机械","limit":20,"type":"mod"}

按英文 / 中文 / slug 匹配（精确 > 前缀 > 包含）。可选 type 限定分类。

分类浏览

GET https://search.mcdb.astral.fan/v1/browse?type=mod&page=0&limit=50

按英文标题字母序分页返回某分类下的条目（供 Web 目录与客户端浏览）。

数据源

AstralNext/MCDB · 每小时同步
