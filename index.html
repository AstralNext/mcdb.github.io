MCDB 汉化 API

静态分片 · 无服务端 · 供启动器按需查询中文标题/简介

Endpoints

  • api/v1/manifest.json
  • api/v1/i18n/{hex(utf8(id[0:2]))}.json — 例：4141.json（AA）

示例

const id = "AANobbMI";
const hex = [...new TextEncoder().encode(id.slice(0, 2))]
  .map(b => b.toString(16).padStart(2, "0")).join("");
const shard = await fetch(`api/v1/i18n/${hex}.json`).then(r => r.json());
console.log(shard[id]?.zh); // 钠

源数据来自 AstralNext/MCDB · 向量检索另建服务
